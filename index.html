Python и Алгоритмы

Введение в Python

Python - это мощный и гибкий язык программирования, который используется для разработки веб-приложений, анализа данных, искусственного интеллекта и многого другого. Одним из главных преимуществ Python является его простота и читабельность, что делает его отличным выбором для начинающих программистов.

Алгоритмы на Python

Алгоритмы - это пошаговые инструкции для выполнения задач или решения проблем. Python предоставляет богатый набор библиотек и инструментов для реализации алгоритмов. Рассмотрим несколько примеров.

Сортировка пузырьком (Bubble Sort)

Сортировка пузырьком - это простой алгоритм сортировки, который многократно проходит через список, сравнивая соседние элементы и меняя их местами, если они находятся в неправильном порядке.

def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

Бинарный поиск (Binary Search)

Бинарный поиск - это эффективный алгоритм для поиска элемента в отсортированном массиве. Алгоритм работает путем многократного деления диапазона поиска пополам.

def binary_search(arr, x):
    low = 0
    high = len(arr) - 1
    mid = 0

    while low <= high:
        mid = (high + low) // 2

        if arr[mid] < x:
            low = mid + 1
        elif arr[mid] > x:
            high = mid - 1
        else:
            return mid

    return -1
